## 2. Why this note exists

Recent probe work established two distinct classes of apparent basin instability:

### A. Measurement-conditioned instability
This occurs when between-cohort spectral comparison appears inconsistent across scale only because the comparison posture is uncalibrated.

This class is resolved by calibrated spectral comparison.

### B. Lawful within-cohort fragmentation
This occurs when a cohort splits into more than one basin under a particular support horizon even after calibrated comparison is applied.

This class is not threshold artifact.
It is a real structural phenomenon within the current Door One observational frame.

Without an explicit note, these two classes can be confused.

---

## 3. Basin interpretation rule

Any apparent basin instability must now be classified as one of:

1. **threshold artifact**
2. **support-horizon resonance**
3. **lawful fragmentation / consolidation**
4. **unresolved cause**

This replaces the weaker earlier habit of treating all instability as generic fragmentation.

---

## 4. Threshold artifact

A basin-facing effect should be classified as threshold artifact when:

- between-cohort separability fails under raw spectral comparison,
- but becomes stable under calibrated comparison,
- and no comparable within-cohort splitting remains once measurement inconsistency is removed.

Working meaning:

The identity boundary was present.
The measurement space was inconsistent.

---

## 5. Support-horizon resonance

A basin-facing effect should be classified as support-horizon resonance when:

- calibrated comparison preserves between-cohort identity,
- but a cohort still splits only at a particular scale,
- and that scale is meaningfully related to the dominant recurrence period.

Working meaning:

The observation horizon is not merely coarse or fine.
It is interacting lawfully with the source’s recurrence structure.

This is a structural explanation, not a semantic one.

---

## 6. Lawful fragmentation / consolidation

A basin-facing effect should be classified as lawful fragmentation / consolidation when:

- splitting or consolidation persists after calibration,
- the effect changes systematically across declared scales,
- and the result remains attributable to structural variation rather than threshold placement.

Working meaning:

The cohort is not failing.
Different support horizons are exposing different organizational expressions of the same lawful source family.

---

## 7. Unresolved cause

A basin-facing effect remains unresolved when:

- it persists after calibration,
- but does not yet map clearly to support-horizon resonance,
- harmonic arrangement,
- or another declared structural explanation.

This category is important because it preserves honesty.
Not every remaining effect should be explained prematurely.
