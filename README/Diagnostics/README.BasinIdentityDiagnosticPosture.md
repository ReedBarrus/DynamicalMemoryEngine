## 2. Why this note exists

Recent probe work has progressively separated several distinct classes of apparent basin instability.

Earlier, the main distinction was:

### A. Measurement-conditioned instability
This occurs when between-cohort spectral comparison appears inconsistent across scale only because the comparison posture is uncalibrated.

### B. Lawful structural residue
This occurs when some splitting or consolidation remains after calibration and therefore reflects a real structural phenomenon rather than threshold artifact.

That first distinction remains valid, but the active probe line has now sharpened the lawful structural residue into additional explanatory layers.

Without an explicit note, the project risks collapsing all remaining instability into generic “fragmentation,” which is now too coarse.

---

## 3. Basin interpretation rule

Any apparent basin instability must now be classified through the following stricter ladder:

1. **threshold artifact**
2. **support-horizon resonance**
3. **exact-edge bifurcation**
4. **active interaction zone**
5. **oscillatory exchange**
6. **lawful fragmentation / consolidation**
7. **unresolved cause**

This replaces the weaker earlier habit of treating all instability as generic fragmentation.

---

## 4. Threshold artifact

A basin-facing effect should be classified as threshold artifact when:

- between-cohort separability fails under raw spectral comparison,
- but becomes stable under calibrated comparison,
- and no comparable within-cohort splitting remains once measurement inconsistency is removed.

Working meaning:

The identity boundary was present.
The measurement space was inconsistent.

---

## 5. Support-horizon resonance

A basin-facing effect should be classified as support-horizon resonance when:

- calibrated comparison preserves between-cohort identity,
- but a cohort still splits only at a particular scale,
- and that scale is meaningfully related to the dominant recurrence period.

Working meaning:

The observation horizon is interacting lawfully with the source’s recurrence structure.

This remains a structural explanation, not a semantic one.

---

## 6. Exact-edge bifurcation

A basin-facing effect should be classified as exact-edge bifurcation when:

- support-horizon resonance is present,
- splitting appears sharply at an exact boundary placement,
- nearby off-edge placements show stress or redistribution but do not split,
- and the response is better described as a sharp edge-conditioned event rather than a broad boundary zone.

Working meaning:

The system is responding not merely to “near-boundary” geometry, but to a sharp interface condition in the active band-profile space.

---

## 7. Active interaction zone

A basin-facing effect should be classified as an active interaction zone effect when:

- a formal edge exists,
- but splitting appears only when cross-boundary redistribution is structurally available across the adjacent bands,
- and inert-edge cases remain consolidated despite formal edge coincidence.

Working meaning:

A boundary is not splitting-prone merely because it exists.
It becomes active only when the adjacent bands form a live redistribution interface.

Important refinement:
not every adjacent-band occupancy pattern is equivalent.
The receiving side of the boundary matters.

---

## 8. Oscillatory exchange

A basin-facing effect should be classified as oscillatory exchange when:

- an active interaction zone exists,
- cross-boundary redistribution is not merely present but organized,
- and the boundary-adjacent band energies exhibit a stable alternating temporal regime across windows.

Current probe language for this is:

- strong oscillatory flow strength,
- strong lag-1 anti-correlation or equivalent period-2 alternation structure,
- temporally bimodal band-profile organization across windows.

Working meaning:

Basin splitting is not being driven merely by static spectral arrangement or one-way drift.
It is being driven by a stable temporal exchange pattern across an active boundary.

This is the strongest current explanation for the active splitting cases.

---

## 9. Lawful fragmentation / consolidation

A basin-facing effect should be classified as lawful fragmentation / consolidation when:

- splitting or consolidation persists after calibration,
- the effect changes systematically across declared scales or boundary conditions,
- and the result remains attributable to structural variation rather than threshold placement.

Working meaning:

The cohort is not failing.
Different lawful observation conditions are exposing different stable organizational expressions of the same source family.

This category should now usually be reached only after testing the stricter explanatory layers above it.

---

## 10. Unresolved cause

A basin-facing effect remains unresolved when:

- it persists after calibration,
- but does not yet map clearly to:
  - support-horizon resonance,
  - exact-edge bifurcation,
  - active interaction zone,
  - oscillatory exchange,
  - or another declared structural explanation.

This category remains important because it preserves honesty.
Not every remaining effect should be explained prematurely.

## 13. Working principle

A basin boundary is not automatically more real because it appears.
A basin instability is not automatically more serious because it fragments.

The first question is still:

**Was the measurement space lawful?**

After that, the next questions are now:

1. is the effect support-horizon dependent?
2. is the effect exact-edge conditioned?
3. is the boundary active or inert?
4. is the remaining splitting driven by oscillatory exchange?
5. only then: is this lawful fragmentation / consolidation or still unresolved?

This is the current preferred order of explanation.
