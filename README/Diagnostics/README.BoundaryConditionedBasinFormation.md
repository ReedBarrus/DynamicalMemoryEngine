## 7. Active interaction zone hypothesis

A boundary should be treated as **active** when:
- both adjacent bands participate meaningfully in the evolving energy profile,
- redistribution across the interface is structurally available,
- and that redistribution can vary across windows.

A boundary should be treated as **inert** when:
- the formal edge exists,
- but one side is not meaningfully populated,
- or no meaningful redistribution occurs across the interface.

This is the current hypothesis.
It remains to be tested directly.
