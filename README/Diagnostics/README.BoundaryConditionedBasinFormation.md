## 7. Active interaction zone

A boundary should be treated as **active** when:
- the formal edge exists,
- cross-boundary redistribution is structurally available across the adjacent bands,
- and the adjacent interface is “live” rather than merely present by formal geometry.

A boundary should be treated as **inert** when:
- the formal edge exists,
- but redistribution is negligible or structurally unavailable,
- or one side of the interface is not participating meaningfully enough to support splitting.

Important refinement:
not every adjacent-band occupancy pattern is equivalent.
The receiving side of the boundary matters.

This means active interaction is not merely:
- “some energy exists nearby.”

It is:
- “cross-boundary redistribution is structurally available in the actual band-profile space.”
