## 6. What the three-gate model proved

The current three-gate model is:

- operationally coherent
- non-redundant
- and sufficient for the current trial range

### Legitimacy
filters whether a claim is trustworthy enough to review at all

### Utility
filters whether promotion would actually compress lawful reuse

### Governability
filters whether the promoted claim would behave safely as a standing reference surface

The most important result is that governability is doing unique work.

Packet 3 in particular shows:
- legitimacy and utility are not enough by themselves
- governability must be able to stop promotion when a claim would otherwise become too sticky, too overread, or too semantically gravitational
